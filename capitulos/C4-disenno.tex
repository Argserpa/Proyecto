Estas son los criterios seguidos para la selección del programa a utilizar para el desarrollo de este TFG.
\begin{itemize}
	\item Capacidad de Streaming: ¿Soporta Live Streaming (en vivo) y Video On Demand (VOD) (grabado)?
	
	\item Protocolos Soportados: RTMP (ingesta), HLS y DASH (distribución a clientes), WebRTC (baja latencia).
	
	\item Transcodificación/Transmuxing: ¿Puede convertir videos a diferentes calidades y formatos sobre la marcha?
	
	\item Escalabilidad: ¿Cómo se escala en un entorno distribuido (contenedores, Kubernetes)? ¿Soporta balanceo de carga?
	
	\item Documentación y Comunidad: Facilidad para encontrar ayuda y recursos.
	
	\item Licenciamiento/Costo: ¿Es de código abierto o comercial?
	
	\item Soporte de Contenedores: ¿Hay imágenes Docker oficiales o bien mantenidas? ¿Ejemplos de despliegue en Kubernetes?
\end{itemize}
